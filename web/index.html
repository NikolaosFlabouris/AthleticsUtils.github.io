About Athletics Utilities

Athletics Utilities provides quick access to World Athletics scoring tables and helps athletes and coaches analyze performance data. All tools work offline once loaded and can be installed as a Progressive Web App on your device.

Key Features:

  • Instant point lookups for any performance
  • Find equivalent performances across events
  • Works completely offline once loaded
  • Install as an app on your device
  • Fast and lightweight - no framework overhead
  • Regular updates with new tools and features
