How to Use

  • Select Gender: Choose between Men's or Women's events
  • Select Combined Event: Choose from Pentathlon, Heptathlon, or Decathlon
  • Enter Performances: Input performance for each discipline in the following formats:
    • Track Events: Enter time as seconds (e.g., "10.5") or minutes:seconds (e.g., "1:23.4" or "4:15.32")
    • Field Events: Enter distances/heights in meters (e.g., "7.50" for long jump, "2.10" for high jump)
    • Hand Timing: For applicable track events, check the hand timing box to add the official offset (0.24s or 0.14s)
  • View Scores: Individual event scores and running totals update automatically as you enter performances
  • Clear All: Use the "Clear All" button to reset all entries and start over

About Combined Events

Combined events are athletics competitions that consist of multiple track and field disciplines. Athletes earn points based on their performance in each event, with the winner being determined by the highest total score.

Scoring System

Each discipline uses World Athletics' official scoring formulas to convert performances into points:

  • Track Events: Points = a × (b - Time)c
  • Jumps: Points = a × (Height - b)c
  • Throws: Points = a × (Distance - b)c

The formulas are calibrated to balance scoring across different event types, ensuring that no single discipline dominates the total score. All point values are rounded down to the nearest whole number.
